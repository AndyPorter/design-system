Accessibility statement for the Design system of HM Land Registry

This website is run by the Design practice of HM Land Registry. We want as many people as possible to be able to use this website. For example, that means you should be able to:

  • change colours, contrast levels and fonts
  • zoom in up to 200% without the text spilling off the screen
  • navigate most of the website using just a keyboard
  • navigate most of the website using speech recognition software
  • listen to most of the website using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)

We've also made the website text as simple as possible to understand.

AbilityNet has advice on making your device easier to use if you have a disability.

How accessible this website is

We intend this site to be fully accessible, but know some parts of may indicate as not fully accessible:

  • Some pages have repeated links to the same location. These are spread out in the page, and sometimes have different link text. Automated tests flag these up as potential issues, but we feel they are ok given the context. We'd be keen to understand if they do cause practical issues

Feedback and contact information

If you need information on this website in a different format, or have suggestions of how we can improve this site:

  • email: andy.porter@landregistry.gov.uk

We'll consider your request and get back to you within 5 days.

Reporting accessibility problems with this website

We're always looking to improve the accessibility of this website. If you find any problems that aren't listed on this page or think we're not meeting the accessibility requirements, email andy.porter@landregistry.gov.uk

Enforcement procedure

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018 (the 'accessibility regulations'). If you’re not happy with how we respond to your complaint, contact the Equality Advisory and Support Service (EASS).

Contacting us by phone or visiting us in person

Currently we only provide a first contact via email. This site is operated by a small team and meant primarily for employees of HM Land Registry.

We will be happy to try to communicate with you by phone, or other means. Please indicate in your first email how we can best communicate further with you.

Technical information about this website’s accessibility

The Design practice in HM Land Registry is committed to making its website accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.

This website is fully compliant with the Web Content Accessibility Guidelines version 2.1 AA standard.

Non accessible content

We have, for this small, limited audience site, not conducted full accessibility testing of everything. There may be items that do not meet the WCAG 2.1 AA standard, that have yet to be tested.

The web pages that make up this site pass the WCAG 2.1 AA standard, but the site contains links to the following items which have not yet been tested by us:

  • Google documents, sheets and surveys
  • a Microsoft Streams hosted video. This is not essential content
  • a PDF of a Video transcript, provided to us. This is not essential content and most of the transcript has been repeated in html in the site

Feedback about the accessibilty of these items, in practical use, is very welcome.

How we tested this website

This website was last tested on 30 October 2020. Tests were carried out by Andy Porter, using Wave, web accessibilty testing tool -a Google Chrome browser plugin, running the website in Google Chrome browser.

  • We tested our main website, and all the top level links from the main page, available at https://design-system-hmlr.herokuapp.com/

What we're doing to improve accessibility

This statement was prepared on 30 October 2020. It was last updated on 30 October 2020.
